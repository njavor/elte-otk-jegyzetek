A Körmozgás


    ask bolygók
    [
        set szög (szög + sebesség) mod 360

        let k középpontneve
        let kpx [xcor] of one-of bolygók with [label = k]
        let kpy [ycor] of one-of bolygók with [label = k]
        
        set xcor kpx + sugár * cos szög
        set ycor kpy + sugár * sin szög
    ]
